# 프로젝트<br>

### 트렐로 사이트 예제<br>

현재 실존하는 사이트인 트렐로의 기능을 축소화한 예제 사이트<br>

![image](https://user-images.githubusercontent.com/62632252/154222005-b49e8f9e-0651-40ae-9d7d-8afce643af4d.png)
# Skills

1. database : 인프런 강사가 직접 제공
2. js framework : Vue.js
3. style : CSS
4. ui framework : Bootstrap

# 주요 기능

1. 권한에 따른 로그인
2. 보드 목록 조회/생성/수정/삭제/셋팅(색상 입히기)
3. 리스트 생성/수정/삭제/이동
4. 카드 목록 조회/생성/수정/삭제/이동

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

For detailed explanation on how things work, consult the [docs for vue-loader](http://vuejs.github.io/vue-loader).
